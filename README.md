# 🌍 Palliative Care Volunteer Coordination Practices

This repository documents how different nations structure, govern, and operate volunteer coordination within hospice and palliative care settings. It is designed to support cross-border learning, policy development, and the creation of localized coordination materials.

## Repository Structure

```
palliative-volunteer-coord/
├── README.md                  # This overview file
├── research-findings.md       # Comprehensive literature review & country practices
├── coordination-models.md     # Comparative models of volunteer coordination
├── training-best-practices.md # Training & onboarding best practices
└── localization/              # Localized materials (translated via Lara Translate)
    └── (language subdirectories)
```

## Key Themes

- Volunteer roles & scope across healthcare systems

- Cultural & socio-economic influences
- Barriers & facilitators to volunteer involvement
